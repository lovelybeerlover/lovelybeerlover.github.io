    Visual Design, Mobile & Desktop

    This project was a collaboration between George Brown College and ClauseHound. I was tasked with converting a non-responsive desktop application to a new, responsive design optimized for mobile usage.











    Background & Challenge

    The desktop application could be divided into thress main sections; A:Additional Clauses B:Clause Examples, and C: Core Elements. On the section 'C', the 'paw' icons appear next to some clauses that are able to be replaced with other clauses. Once replacable clauses are clicked, Section 'A' displays additional clauses that are avaliable for the clicked clause. When each additional clause is clicked, Section'B' displays the examples.
    Since the app is text heavy, I was challenged by the limited screen real estate.
